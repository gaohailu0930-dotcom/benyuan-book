# 本源币：重写经济的地基

> 一部独立完成的经济理论著作，全书免费阅读与传播。

几百年来，有一条假设被所有人继承，却从没有人认真质疑过：**货币，必须为正。**

这本书从这一个念头出发，推演出一套允许账户为负、而全社会总账恒为零的记账体系——**本源币**。它不止步于一个口号：从生产、劳动、环境、公共服务、人口，到创新、信用、日常生活、治理与文明定位，全书十四章，几乎不用一个公式，只用逻辑和日常的语言，讲给每一个普通人听。

— 高海璐 著
